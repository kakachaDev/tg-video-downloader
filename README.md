# YouTube Downloader Bot

Telegram inline-бот для скачивания видео и аудио с YouTube. Работает в любом чате через `@упоминание`.

## Как использовать

В любом чате напишите `@НазваниеБота ссылка` — бот покажет варианты качества:

```
@NyaDownloadBot https://youtu.be/dQw4w9WgXcQ
```

- `⚡` — файл уже скачан, отправится мгновенно
- `⏳` — нужно скачать, займёт время
- `⛔` — превышает лимит Telegram (50 МБ)

После выбора качества в чат прилетает сообщение «Загружаю...», которое заменяется на файл как только скачается.

Поддерживаются обычные ссылки, короткие (`youtu.be/...`) и Shorts (`youtube.com/shorts/...`).

## Требования

- Python 3.12+
- ffmpeg

```bash
sudo apt install ffmpeg
```

## Установка

```bash
git clone <repo>
cd downloader
./setup.sh
```

`setup.sh` создаёт виртуальное окружение, устанавливает зависимости и копирует `.env.example` в `.env`.

## Настройка `.env`

```env
# Токен от @BotFather
BOT_TOKEN=123456789:AAxxxxxx

# ID чата, куда бот временно загружает файлы для получения file_id.
# Создайте приватную группу, добавьте туда бота и узнайте ID командой /chatid.
STORAGE_CHAT_ID=-5251557070

# Ваш Telegram user ID (узнать у @userinfobot).
# Только этот аккаунт может использовать /cookie.
ADMIN_USER_ID=123456789

# Путь к файлу cookies (опционально, по умолчанию cookies.txt в папке бота).
# COOKIES_FILE=cookies.txt
```

## Настройка BotFather

1. `/setinline` — включить inline-режим
2. `/setinlinefeedback` → выбрать бота → **Enabled** — без этого бот не узнает, какое качество выбрано

## Запуск

```bash
./start.sh   # запуск в фоне, с watchdog-перезапуском при падении
./stop.sh    # остановка
tail -f bot.log  # логи в реальном времени
```

`start.sh` запускает бота в цикле: если процесс упадёт — автоматически перезапустится через 10 секунд.

## Автозапуск при перезагрузке сервера

```bash
crontab -e
```

Добавить строку:

```
@reboot cd /путь/к/downloader && ./start.sh
```

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Приветствие и инструкция |
| `/chatid` | Показывает ID текущего чата (нужен для настройки `STORAGE_CHAT_ID`) |
| `/cookie` | Обновить `cookies.txt` (только для админа) |

### Обновление cookies

Если YouTube требует авторизацию (ролики 18+, ограниченный доступ):

1. Установите расширение [Get cookies.txt LOCALLY](https://chrome.google.com/webstore/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc)
2. Зайдите в YouTube под своим аккаунтом, экспортируйте `cookies.txt`
3. Отправьте файл боту с подписью `/cookie`

Для публичных видео cookies не нужны.

## Кеш файлов

Бот сохраняет `file_id` уже загруженных файлов в `file_cache.json`. При повторном запросе того же видео в том же качестве файл отправляется мгновенно без повторного скачивания.

## Ограничения

- Максимальный размер файла: **50 МБ** (ограничение Telegram Bot API)
- Варианты качества, превышающие лимит, помечаются `⛔` и недоступны для скачивания

## Структура файлов

```
bot.py            — код бота
requirements.txt  — зависимости Python
setup.sh          — первоначальная установка
start.sh          — запуск / перезапуск
stop.sh           — остановка
.env              — секреты (не коммитить)
.env.example      — шаблон конфига
file_cache.json   — кеш file_id (создаётся автоматически)
bot.log           — логи
bot.pid           — PID фонового процесса
```
